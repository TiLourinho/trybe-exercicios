# Dia 29.1 - MongoDB - Introdução

Neste dia aprendi a:

- Instalar e conectar-se ao **MongoDB**;
- Listar bancos de dados e coleções;
- Inserir documentos no banco de dados usando o _insert()_;
- Pesquisar documentos no banco de dados usando o _find()_;
- Contar a quantidade de documentos retornados usando o _countDocuments()_;
- Filtrar o resultado das suas pesquisas;
- Limitar e pular os documentos retornados;
- Fazer uma paginação simples combinando os métodos _limit()_ e o _skip()_;
- Entender o que é uma projeção e como utilizá-la.
